Introducción a HTML

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Proporciona una forma de definir el contenido, el diseño y la apariencia de una página web.

Estructura básica de HTML

Cada documento HTML tiene una estructura básica que incluye los siguientes elementos:

  • <html> - Define la raíz del documento HTML
  • <head> - Contiene metadatos sobre el documento, como el título
  • <body> - Contiene el contenido visible de la página web

Elementos y etiquetas HTML

Los elementos HTML se definen utilizando etiquetas, que se encierran entre corchetes angulares. Por ejemplo, la etiqueta <h1> se utiliza para crear un encabezado de nivel superior.
